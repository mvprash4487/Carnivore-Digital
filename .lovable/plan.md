

# Carnivore Digital — Ultra-Luxury Rebuild

Rebuild the Carnivore Digital website as a cinematic, ultra-premium digital agency experience. Dark theme with gold accents, smooth animations, and museum-quality design that feels like a luxury fashion house website.

## Design System
- **Dark base**: Near-black (#0A0A0A) with rich charcoal surfaces
- **Gold accent**: Warm gold (#C9A84C) for highlights, borders, and hover states
- **Typography**: Elegant serif headings (Playfair Display), clean sans-serif body (Inter)
- **Animations**: Smooth scroll-triggered fade-ins, parallax effects, subtle hover reveals
- **Spacing**: Generous whitespace — everything breathes luxury

## Sections (Single Page)

### 1. Navigation
- Minimal fixed nav with logo left, links right
- Glassmorphism effect on scroll
- Gold underline on active/hover states

### 2. Hero Section
- Full-viewport cinematic hero with the cocktail glass imagery
- "We'll Devour Your Competition" in oversized elegant serif typography
- Subtle parallax on background image
- Animated gold divider line
- Steve Jobs quote styled as an elegant pull-quote
- Smooth CTA button with gold border and hover fill

### 3. About Section
- Split layout: large editorial number "01" with gold accent
- Text content with refined typography and generous line-height
- "Since 2015" as a highlighted stat with animated counter
- Subtle reveal animation on scroll

### 4. Services Section
- 5 service cards in an elegant grid
- Each card: dark glass card with gold icon, service name, description
- Hover effect: gold border glow + subtle lift
- Services: UX/UI, Digital Marketing, Photo & Video, Branding, Web Design & SEO
- Uses all original copy

### 5. Portfolio / Our Work
- Masonry-style gallery with hover overlays showing project names
- Images sourced from the original site (Ocean, Thara Thong, Burger & Lobster, Canolini, ROSH)
- Smooth image scale on hover with gold overlay text
- Section intro with editorial numbering "03"

### 6. Clients Section
- Elegant horizontal logo marquee with grayscale-to-color on hover
- Client logos: Ocean, Radisson Blu, Royal Orchid Sheraton, Aspira, Getfresh, Marriott Bangkok
- Gold divider accents

### 7. Contact / Footer
- Luxurious CTA section: "Let's Create Something Extraordinary"
- Minimal contact form or prominent "Let's Connect" button
- Footer with social links and copyright

## Interactions & Polish
- Scroll-triggered animations using Framer Motion (fade-up, stagger reveals)
- Smooth scrolling between sections via nav links
- Custom cursor effect (optional gold dot follower)
- Responsive design — equally stunning on mobile
